# H5-Canvas-Game-Square
一个自己写的canvas小游戏。
>整个游戏过程比较简单，玩家只需要点击手机屏幕来使中间粉红的那个小方块跳动起来便可，如果小方块被带到了屏幕的左边，那么游戏结束；在屏幕的最中>间有一个时间计数器，每隔10s，计数器上的数字变换一次颜色，玩家的方块不能踩在与计数器相同颜色的矩形方块上，一旦掉下去，那么游戏结束.
